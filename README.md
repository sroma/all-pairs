# all-pairs
All-pairs shortest-path problem in directed weighted graph (parallel version).

floyd-warshall.cpp - realization of Floyd-Warshall algorithm (C++/OpenMP). Graph is stored as connectively matrix.

johnson.cpp - realization of Johnson algorithm (C++/OpenMP). Dijkstra  algorithm is realized on arrays. Graph is stored as array of edges.

graph_search_main.cpp - Test program for Floyd-Warshall alg. & Jonhson alg. with variations of Dijkstra (set or array). 

Test case: Rome graph from http://www.dis.uniroma1.it/~challenge9
